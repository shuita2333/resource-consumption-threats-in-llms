# Interpretability

This page collects papers that help explain, analyze, or interpret resource consumption behavior in large models and agent systems.

## Included Scope

- Resource usage attribution
- Bottleneck analysis
- Mechanistic understanding of compute or memory overhead
- Interpretable profiling of inference or agent workflows
